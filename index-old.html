SMKN 1 Soreang

Pameran teaching factory tahun 2019 yang diselenggarakan SMKN 1 Soreang, yang dihadiri oleh cabang dinas pendidikan provinsi Jawa Barat. Produk yang saya tampilkan pada pameran ini adalah Absen digital. Absen digital bertujuan untuk menghitung jumlah absen siswa. Siswa hanya perlu menempelkan kartu RFID ke mesin absen sebagai alat absen.

Untuk membuat produk ini ada beberapa kemampuan yang saya gunakan antara lain:

  • C
  • PHP
  • HTML
  • CSS
  • HTTP
  • Apache
  • Mysql
